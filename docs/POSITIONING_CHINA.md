# UnodeAi — 定位与中国市场价值（给 Vincy 的参考）

> 给新同事的一页纸：UnodeAi 是什么、最大卖点、在中国为什么有价值、和谁竞争、以及要赢必须认清的风险。
> 这是**观点性策略文档**(2026-06-16)，不是产品规格；规格见 PRD/STATUS。诚实优先,不吹。

## 0. 一句话：UnodeAi 是什么
一个 VS Code 扩展，让你在编辑器里**组建一支 AI 研发团队**(PM + 架构/开发/测试/审查…)，PM 拆活、分派、并行执行、**验证通过才合并**。每个角色可用**不同(且便宜)的模型**，默认走 Roam(unode) OpenAI 兼容网关，50+ 模型、折扣价。
不是"更聪明的单个助手"，是"**一条用便宜模型也能稳定交付的研发流程**"。

## 1. 最大卖点（整体）
**"一支你养得起的、可验证的 AI 研发团队。"** 三层缺一不可：
1. **编排式多 agent**：PM 委派 + 专家分工 + worktree 隔离 + 自动合并。单 agent 工具(Cline/Cursor/Copilot)做不到。
2. **验证门禁 (verifier-as-gate)**：只有跑过你测试的代码才进主线。"多 agent"别人能抄，"**只合并验证通过的工作**"是把可靠性做成机制——这是真正的护城河。
3. **弱模型 + 强框架 + 成本套利**：按角色分派便宜模型，用框架(验证门禁、给弱模型强制 XML 工具协议、写文件防灾、项目约定注入)把可靠性补上。跑一整队的钱，可能比别人跑一个前沿模型还省。

## 2. 在中国的特殊价值（被低估，可能是最强的地方）
中国市场把上面的卖点放大成近乎"天选契合"：

1. **国产模型生态的最佳 agent 载体。** 中国有 DeepSeek / Kimi(Moonshot) / Qwen(通义) / GLM(智谱) / MiniMax——**便宜、越来越强、国内可直连、可用人民币付费**。短板是 **agentic 工具调用偏弱**(会把工具调用当文本吐出来、卡死)。Roam 的框架正好补这块(验证门禁、对 leaker 强制 XML、防灾、约定注入)。→ Roam 可定位为"**让国产便宜模型也能可靠组队干活的那一层**"，这是竞品没在做的。

2. **成本敏感。** 国内团队对价格极敏感。"按角色用便宜模型 + 折扣网关(group_ratio)"直接命中;前沿模型工具又贵又难付。

3. **可及性 / 合规——最致命也最被忽视。** Cursor / Copilot / Claude Code 默认依赖**被墙、难付费、ToS 受限**的境外模型，国内体验是残的。Roam **默认国产模型 + 境内可用网关**，从根上绕开。竞品"装了也用不顺"，Roam"开箱即用、能正常付费"——这本身就是壁垒。

**中国定位一句话**：**"用国产便宜模型，组一支在国内能正常用、正常付费、还跑得起的可靠 AI 研发团队。"**

## 3. 竞争格局与差异化
| 对手 | 它是什么 | Roam 的差异化 |
| --- | --- | --- |
| Cline / Roo / Cursor / Copilot | 单 agent 助手，默认绑境外前沿模型 | 多 agent + 验证合并 + 自带任意便宜模型 + 国内可用 |
| Kilo Code | 有并行 worktree，但无协调/验证 | **编排式 + 验证门禁 + 自动合并**(不是裸并行) |
| 国内大厂(通义灵码 / 百度 Comate / 字节 Trae·MarsCode / 腾讯) | 免费、深度集成、绑自家模型的单 agent 助手 | 唯一能赢的姿势是**那套组合**:多 agent 编排 + 验证 + 自带便宜模型 + 成本套利 + 自建 agent/技能。**单点打不过免费大厂,组合才有戏。** |

## 4. 诚实的风险（要赢必须认清）
- **可靠性还在加固。** 真机 dogfooding 证明:弱模型上核心闭环仍脆(卡死、幻觉、误拦)。卖点是**架构方向**对,但执行(0.9 弱模型加固)必须追上,否则商用用户首小时就流失。**这就是 0.9 加固→再上 1.0 商用的原因。**
- **国内大厂在卷且免费。** 必须靠组合差异化,不能拼单点。
- **网关是命脉。** 成本套利/折扣/国内可付费都押在 unode 网关上;它的稳定性、SLA、合规 = 产品的稳定性。
- **诚信底线(对内对外都适用)。** PM/agent 曾对用户"睁眼说瞎话"(凭记忆报错版本号)。商用产品里,协调员对用户撒谎比 worker 出错更伤信任——0.9 在修这个(强制读后再答 + 让 PM 用更强/更诚实的模型)。

## 5. 给市场/商务的话术要点（事实导向）
- 主打**结果**不主打机制:"省钱、可靠交付、国内能用"。
- 50+ 模型 / 独家折扣 / 可靠 SLA(SLA 暂用软话术,别过度承诺)。
- 别把未发布/实验功能说成已支持(诚信底线,也是 1.0 商用信任的基础)。
- 商店页/README 暴露用户手册(USAGE)与 wiki 链接,降低上手门槛。

---
**给 Vincy 的 TL;DR**：Roam 不是"更强的 AI 助手",是"**用便宜国产模型组一支验证过、能自动合并、国内可正常付费使用的 AI 研发团队**"。最大机会在中国(模型便宜可及+团队省钱),最大风险是可靠性还在追(0.9)和大厂免费内卷——所以我们靠"编排+验证+自带便宜模型+自建 agent"的组合,而不是拼单一模型或单点功能。
